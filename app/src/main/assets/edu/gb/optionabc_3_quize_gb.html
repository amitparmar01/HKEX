1. 理论上，认购期权持有人的潜在收益是 _______ 的，而认购期权沽出者的潜在收益是 ______ 的。

    1. 有限；有限 
    2. 有限；无限 
    3. 无限；有限 
    4. 无限；无限 
    正确答案是 "C"

    解释：

    理论上，只要正股的股价上升，认购期权持有人的潜在收益是无限的。不过，认购期权沽出者的潜在收益则只限于所收到的期权金。

  2. 下列哪一项期权持仓在行使或指定分配后须购入正股?

    I. 买入认购

    II. 买入认沽

    III. 沽出认购

    IV. 沽出认沽

    1. I 和 II 
    2. I 和 IV 
    3. II 和 III 
    4. II 和 IV 
    正确答案是 "B"

    解释：

    认购期权的持有者有权以行使价购入正股；认沽期权的沽出者则有责任在期权被行使时以行使价购入正股。

  3. 如果认沽期权长仓的持有者在行使期权时获得利润，有关正股的价格必定是：

    1. 高于行使价 
    2. 高于行使价加上已付出的期权金 
    3. 低于行使价 
    4. 低于行使价减去已付出的期权金 
    正确答案是 "D"

    解释：

    对认沽期权的持有人来说，打和点是期权的行使价减去已付出的期权金。

  4. 认沽期权的卖方（沽出者）：

    1. 有权以行使价沽出正股 
    2. 有权以行使价购入正股 
    3. 有责任在认沽期权被行使时以行使价购入正股 
    4. 有责任在认购期权被行使时以行使价沽出正股 
    正确答案是 "C"

    解释：

    该期权沽出者收取期权金以换取该认沽期权的责任。如期权买方行使其权利以行使价沽出正股，期权卖方便有责任以行使价向期权买方购入正股。

  5. 以下哪一个 / 几个陈述是正确的?

    I. 期权合约持有者可收取期权金

    II. 期权合约沽出者有责任购入或沽出正股

    III. 期权合约持有人所面对的风险是无限的

    IV. 期权合约沽出者的潜在收益是无限的

    1. 只是 I 
    2. 只是 II 
    3. I 和 III 
    4. III 和 IV 
    正确答案是 "B"

    解释：

    如认购期权被行使，期权沽出者有责任沽出正股；如认沽期权被行使，期权沽出者有责任购入正股。
