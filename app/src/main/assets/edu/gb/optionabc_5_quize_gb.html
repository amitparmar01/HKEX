1. 随着认沽期权的引伸波幅增加，此认沽期权的期权金一般会：

    1. 增加 
    2. 减少 
    3. 维持不变 
    4. 无法确定 
    正确答案是 "A"

    解释：

    当认沽及认购期权的波幅增加时，这两种期权的期权金亦会增加。 波幅愈大，即期权成为「价内」的可能性愈高。

  2. 正股价上升时，假设其它因素不变，认沽期权的期权金：

    1. 一般会增加 
    2. 一般会减少 
    3. 会维持不变 
    4. 无法厘定 
    正确答案是 "B"

    解释：

    正股的价格有变时，期权金亦会变动。如正股价格上升，认沽期权的期权金会因为内在值的下跌而减少。

  3. 如正股的市价 _______ ，认购期权便属「价外」

    1. 等于行使价 
    2. 高于行使价 
    3. 低于行使价 
    4. 等于行使价加上或减去期权金 
    正确答案是 "C"

    解释：

    根据定义，当正股的市价低于行使价，期权金已无内在值时，认购期权便属「价外」。

  4. XYZ的正股现价是70元。XYZ四月份60元认沽期权目前的期权金为1.5元。这些认沽期权的内在值是多少?

    1. 0元 
    2. 1.5元 
    3. 10元 
    4. 无法计算 
    正确答案是 "A"

    解释：

    内在值是期权行使价与正股价之差额。不过，只有价内期权才有内在值。这些认沽期权已属「价外」，其内在值就是零。

  5. 如果你想在平稳市中得益，应采取以下哪些策略?

    I. 买入认购

    II. 买入认沽

    III. 沽出认购

    IV. 沽出认沽

    1. I 和 II 
    2. I 和 IV  
    3. II 和 III 
    4. III 和 IV 
    正确答案是 "D"

    解释：

    在市况平稳时，期权金会随时间流逝而减少。此乃买入认购或买入认沽期权者必须付出的代价，亦为沽出期权者的收入来源。

  6. 期权价值的时间损耗对哪一方有利

    1. 期权持有者 
    2. 期权沽出者 
    3. 市场庄家 
    4. 以上各项均是 
    正确答案是 "B"

    解释：

    时间损耗不利于期权持有者，但有利于期权沽出者。距离到期的时间愈少，期权到期时变成「价内」的可能性亦愈少。
