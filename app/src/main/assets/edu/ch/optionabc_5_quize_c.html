1. 隨著認沽期權的引伸波幅增加，此認沽期權的期權金一般會：

    1. 增加 
    2. 減少 
    3. 維持不變 
    4. 無法確定 
    正確答案是 "A"

    解釋：

    當認沽及認購期權的波幅增加時，這兩種期權的期權金亦會增加。 波幅愈大，即期權成為「價內」的可能性愈高。

  2. 正股價上升時，假設其他因素不變，認沽期權的期權金：

    1. 一般會增加 
    2. 一般會減少 
    3. 會維持不變 
    4. 無法釐定 
    正確答案是 "B"

    解釋：

    正股的價格有變時，期權金亦會變動。如正股價格上升，認沽期權的期權金會因為內在值的下跌而減少。

  3. 如正股的市價 _______ ，認購期權便屬「價外」

    1. 等於行使價 
    2. 高於行使價 
    3. 低於行使價 
    4. 等於行使價加上或減去期權金 
    正確答案是 "C"

    解釋：

    根據定義，當正股的市價低於行使價，期權金已無內在值時，認購期權便屬「價外」。

  4. XYZ的正股現價是70元。XYZ四月份60元認沽期權目前的期權金為1.5元。這些認沽期權的內在值是多少?

    1. 0元 
    2. 1.5元 
    3. 10元 
    4. 無法計算 
    正確答案是 "A"

    解釋：

    內在值是期權行使價與正股價之差額。不過，只有價內期權才有內在值。這些認沽期權已屬「價外」，其內在值就是零。

  5. 如果你想在平穩市中得益，應採取以下哪些策略?

    I. 買入認購

    II. 買入認沽

    III. 沽出認購

    IV. 沽出認沽

    1. I 和 II 
    2. I 和 IV  
    3. II 和 III 
    4. III 和 IV 
    正確答案是 "D"

    解釋：

    在市況平穩時，期權金會隨時間流逝而減少。此乃買入認購或買入認沽期權者必須付出的代價，亦為沽出期權者的收入來源。

  6. 期權價值的時間損耗對哪一方有利

    1. 期權持有者 
    2. 期權沽出者 
    3. 市場莊家 
    4. 以上各項均是 
    正確答案是 "B"

    解釋：

    時間損耗不利於期權持有者，但有利於期權沽出者。距離到期的時間愈少，期權到期時變成「價內」的可能性亦愈少。
