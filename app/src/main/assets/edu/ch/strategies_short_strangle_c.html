沽出勒束式組合

返回
策略
沽出勒束式組合
成份
沽出相同月份，較低行使價/水平的認沽期權及較高行使價/水平的認購期權
潛在盈利
  • 當正股價/指數水平介乎高低兩個打和點之間
  • 只限於已收取的期權金總額
最大虧損
  • 當正股價/指數水平跌破較低打和點，極大及等如較低打和點減去正股價/指數水平
  • 當正股價/指數水平升越較高打和點，無限及等如正股價/指數水平減去較高打和點
時間值影響
正面
打和點
  • 較低打和點：較低行使價/水平減去已收取期權金總額
  • 較高打和點：較高行使價/水平加上已收取期權金總額
備註
與馬鞍式組合短倉相比，勒束式組合短倉收取的期權金較少，但正股價/指數水平在相對較大幅度上落時才會出現虧損。

成份
沽出「ABC 六月180元認沽」，收取 5元，同時沽出「ABC六月200元認購」，收取 10元
期權金淨額
收取 5元 + 10元 = 15元
打和點
  • 較低打和點：180元–15元 = 165元
  • 較高打和點：200元 + 15元 = 215元
錄得盈利
當正股價介乎 165元及 215元
潛在盈利
15元
潛在虧損
  • 當正股價跌破165元：165元–正股價
  • 當正股價升越215元：正股價–215元
