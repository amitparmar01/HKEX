1. 理論上，認購期權持有人的潛在收益是 _______ 的，而認購期權沽出者的潛在收益是 ______ 的。

    1. 有限；有限 
    2. 有限；無限 
    3. 無限；有限 
    4. 無限；無限 
    正確答案是 "C"

    解釋：

    理論上，只要正股的股價上升，認購期權持有人的潛在收益是無限的。不過，認購期權沽出者的潛在收益則只限於所收到的期權金。

  2. 下列哪一項期權持倉在行使或指定分配後須購入正股?

    I. 買入認購

    II. 買入認沽

    III. 沽出認購

    IV. 沽出認沽

    1. I 和 II 
    2. I 和 IV 
    3. II 和 III 
    4. II 和 IV 
    正確答案是 "B"

    解釋：

    認購期權的持有者有權以行使價購入正股；認沽期權的沽出者則有責任在期權被行使時以行使價購入正股。

  3. 如果認沽期權長倉的持有者在行使期權時獲得利潤，有關正股的價格必定是：

    1. 高於行使價 
    2. 高於行使價加上已付出的期權金 
    3. 低於行使價 
    4. 低於行使價減去已付出的期權金 
    正確答案是 "D"

    解釋：

    對認沽期權的持有人來說，打和點是期權的行使價減去已付出的期權金。

  4. 認沽期權的賣方（沽出者）：

    1. 有權以行使價沽出正股 
    2. 有權以行使價購入正股 
    3. 有責任在認沽期權被行使時以行使價購入正股 
    4. 有責任在認購期權被行使時以行使價沽出正股 
    正確答案是 "C"

    解釋：

    該期權沽出者收取期權金以換取該認沽期權的責任。如期權買方行使其權利以行使價沽出正股，期權賣方便有責任以行使價向期權買方購入正股。

  5. 以下哪一個 / 幾個陳述是正確的?

    I. 期權合約持有者可收取期權金

    II. 期權合約沽出者有責任購入或沽出正股

    III. 期權合約持有人所面對的風險是無限的

    IV. 期權合約沽出者的潛在收益是無限的

    1. 只是 I 
    2. 只是 II 
    3. I 和 III 
    4. III 和 IV 
    正確答案是 "B"

    解釋：

    如認購期權被行使，期權沽出者有責任沽出正股；如認沽期權被行使，期權沽出者有責任購入正股。
